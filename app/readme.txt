Написать андроид приложение с использованием API http://jsonplaceholder.typicode.com.
Использовать нужно только один метод http://jsonplaceholder.typicode.com/posts
Главный экран приложения отображает список постов с заголовком и кратким описанием
(до 100 символов). Также у каждого поста есть индикатор добавленого в избранное
(звездочка активная/неактивная).
По нажатии на элемент списка открывается экран с детальным описанием поста. Экран
содержит заголовок, полный текст и кнопку добавления в избранное. 
При удалении поста из избранного сначала должен показываться диалог с текстом
"Удалить из избранного" и кнопками Да/Нет. По нажатии на "Да" происходит удаление.

Данные запроса должны кешироваться в БД и использоваться в случае проблем с сетью.
Информация о избранных постах храниться только локально. При входе на экран должен
быть индикатор загрузки данных. В случае ошибки загрузки данных, должна быть
показана ошибка на экране.

Результатом тестового задания должен быть apk файл и проект gradle, который можно
запустить в android studio.
